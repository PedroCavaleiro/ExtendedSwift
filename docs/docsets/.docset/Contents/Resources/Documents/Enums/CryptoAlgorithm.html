CryptoAlgorithm

public enum CryptoAlgorithm

A collection of cryptographic algorithms used for hashing and HMAC operations. This enum provides access to various cryptographic algorithms, including MD5, SHA1, SHA224, SHA256, SHA384, and SHA512. Each case provides the corresponding CCHmacAlgorithm.
